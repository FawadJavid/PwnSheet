## 📖 What is PwnSheet?

A single-file, browser-based command reference for offensive security practitioners. Covers the full penetration testing lifecycle — recon, web attacks, API attacks, shells, Linux/Windows privilege escalation, Active Directory, tunneling, file transfer, hash cracking, and more.

No installation. No backend. No dependencies. Open the file and go.

**330+ commands across 13 categories** — instant search, one-click copy, 6 switchable themes, and quick links to essential resources — all running fully offline.

---

## ⬇️ Usage

Download [`pwn-sheet.html`](pwn-sheet.html) and open it in any browser. That's it.

```bash
git clone https://github.com/YOUR_USERNAME/pwnsheet.git
open pwnsheet/pwn-sheet.html
```

> Commands are pre-filled with descriptive placeholders like `TARGET_IP` and `YOUR_LHOST` — copy any command and replace with your actual values.
