📖 What is PwnSheet?

PwnSheet is an interactive offensive security reference built for speed and efficiency. Commands cover the full penetration testing lifecycle — from reconnaissance and exploitation to privilege escalation and lateral movement. Pre-filled with placeholder values, just replace them with your actual target details. Copy any command and you are ready to go.

⌨ How to Use

Commands are pre-filled with example placeholders like TARGET_IP and YOUR_LHOST. Copy any command and replace these with your actual values.

Use search (Ctrl+K) to filter all commands instantly. Navigate sections with Ctrl+→ / Ctrl+←.
